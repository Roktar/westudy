강원대 자바 웹개발 스터디

스터디리더

윤봉길

7월 말에 점수 내는 것 목표로 단기 스터디 모집합니다.

* 실제시험 경험자로, 점수대는 90~100점대면 좋겠습니다. - 실제 시험경험이나 노하우 등을 공유해보는 것도 좋을 듯합니다. (점수가 없으시더라도, 비슷한 수준에서 서로 도움이 되었으면 좋겠습니다.)

* 실제로 많은 실력향상이 있을 수도 있고, 아닐 수도 있지만, 학원이 아니더라도 서로서로 페이스 메이커가 되어 짧은 시간안에 목표 이상의 성과를 내는 것을 추구합니다.

만약에 스터디가 성사된다면, 기본적으로 빠른 시간내에 점수를 내는 것을 목표로 하기에 조금 타이트한 스케쥴로 진행될 예정입니다.

* 처음에 와서 10분동안 단어시험 * 리스닝 문제 2~3개 같이 풀고 노트 테이킹 비교

* Speaking 1. 표현 암기한 것 시험 (10분) 2. mock-up test (30분) 3. 서로 답안 분석해주기

* Writing 1. 표현 암기한 것 시험 (10분) 2. mock-up test (격일로 통합 / 독립 시험) (30분)

3. 서로 답안 분석해주기

* 예상 비용 : 신촌 스터디룸 이용료 * 디테일한 부분은 협의 후 결정
